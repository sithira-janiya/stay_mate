**StayMate** is a full-stack web application built to digitalize and automate **boarding house, tenant, room, meal, and financial management**.

The system supports **tenants, landlords, meal providers, and administrators**, using **JavaScript-based MERN technologies** alongside **Java-based components** for structured backend logic and system workflows.
